# Agent Team: Niche Market Tool Brainstorming

A multi-agent brainstorming system for exploring passive income tool ideas from different perspectives.

## Team Roles

| Agent | Role | Focus |
|-------|------|-------|
| **Idea Generator** | Creative | Generates innovative tool concepts for niche markets |
| **Market Analyst** | Business | Evaluates market potential, competition, monetization |
| **Tech Architect** | Technical | Assesses implementation feasibility and architecture |
| **Devil's Advocate** | Critical | Challenges assumptions, identifies risks and weaknesses |

## Usage

```bash
python run_brainstorm.py "Your niche or problem area"
```

## Example

```bash
python run_brainstorm.py "tools for indie game developers"
```

## Output

The team produces a structured report with:
- Generated ideas ranked by potential
- Market analysis for top ideas
- Technical implementation roadmaps
- Risk assessment and mitigation strategies
