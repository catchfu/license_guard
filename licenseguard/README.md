# LicenseGuard - Agent Team Build

**Product:** License key generation, validation API, and piracy protection for desktop/mobile apps

**Target:** Indie developers selling downloadable software

**Revenue Model:** $15/mo flat or $0.01 per validation

## Team Structure

| Agent | Combined Roles | Responsibility |
|-------|---------------|----------------|
| **Product Lead** | PM + BA | Requirements, roadmap, user stories, priorities |
| **UX & Content Lead** | UX Design + Tech Writer | UI/UX, documentation, API docs, onboarding |
| **Engineering Lead** | Developer + Tech Architect | Code, architecture, implementation |
| **QA Engineer** | QA | Testing, quality assurance, security audit |
| **Growth Lead** | Marketing + Sales | Go-to-market, promotion, pricing strategy |

## Project Phases

1. **Discovery** - Requirements gathering, competitive analysis
2. **Design** - Architecture, UX wireframes, API design
3. **Build** - MVP development (3-4 weeks target)
4. **Launch** - Beta release, documentation, marketing
5. **Iterate** - Feedback loop, improvements

## Usage

```bash
python run_licenseguard_team.py <phase>
```

Phases: `discovery`, `design`, `build`, `launch`, `full`
